
# Changelog

Все заметные изменения в проекте документируются в этом файле.

Формат: семантические версии с кратким списком изменений.

## [Unreleased]

- Работа над AI-отчётами (в процессе): endpoint `/api/report` — заглушка с генерацией PNG-диаграммы, опциональная переадресация запроса к OpenAI при наличии `OPENAI_API_KEY`.

- Добавлен `LICENSE` (MIT).
- Добавлен endpoint `POST /api/transfer` — перевод средств между счетами: создает две транзакции и обновляет балансы.
- Добавлен frontend: формы для управления категориями, создания бюджетов и выполнения переводов между счетами.

---

## [0.1.0] - Initial release

Первоначальная рабочая заготовка проекта. Включено:

- Backend (FastAPI + SQLModel) — модели: `Category`, `Account`, `Transaction`, `Budget`, `PlannedItem`, `Attachment`.
- CRUD-эндпоинты для категорий, счетов, транзакций, бюджетов и планируемых операций.
- Загрузка файлов (`/api/upload`) и отдача загруженных файлов.
- Генерация простых графиков (matplotlib) для отчётов и отдача сохранённых PNG.
- Frontend: минимальный React-приложение (Vite) с PWA-манифестом и service-worker-заготовкой.
- Документация: `README.md` с инструкциями по запуску.
